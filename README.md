チケット管理アプリケーションです。

# リポジトリ

各リポジトリの概要を記載します。
詳細は各リポジトリの README を参照してください。

# server

api を提供します。

# webasset

フロントを提供します。

# webserver

webasset でビルドしたアセットから webserver を起動します。

# search

elasticSearch から検索機能を提供します。

# logstash

> [!Warning]
> 現在は api から同期しているため利用していません。

mysql から elasticSearch にデータを同期します。
